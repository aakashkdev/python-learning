# Python Learning

Is repository me Python ke basic concepts ko examples ke saath explain kiya gaya hai.  
Yeh beginners ke liye helpful hai jo Python ko step-by-step seekhna chahte hain.

## Topics Covered

1. **Variables & Data Types**
2. **Operators**
3. **Conditional Statements (if-else)**
4. **Loops (for, while)**
5. **Functions**
6. **Object-Oriented Programming (OOP)**  
   - Class and Object  
   - Encapsulation  
   - Abstraction  
   - Inheritance  
   - Polymorphism
